Q1 : Lưu dữ liệu trong code

+ Biến : Tham chiếu đến ô nhớ trong ổ cứng ( nơi lưu trữ )
+ KDL : Ràng buộc các giá trị mà các biến được lưu trữ
Vd : int age = 20; // Khai báo và khởi tạo

KNT : 
- Chỉ được phép sử dụng
- Không thể tạo thêm
- Thường bắt đầu bằng chữ cái viết thường

KĐT: 
- Sử dụng các kiểu dữ liệu đối tượng có sẵn của ngôn ngữ 
--> Boolean, Character, Byte, Short, ..
- Tạo thêm để sử dụng 
--> Item ( id, name )
- Giá trị đơn hoặc nhiều tt bên trong
